struct units\+::scaled\+\_\+unit$<$ M, U $>$}
A common point for a hierarchy of units. 

A unit is an entity defined and adopted by convention, with which any other quantity of the same kind can be compared to express the ratio of the second quantity to the first one as a number.

All units of the same dimension can be convereted between each other. To allow this all of them are expressed as different ratios of the same one proprietary chosen reference unit (i.\+e. all length units are expressed in terms of meter, all mass units are expressed in terms of gram, ...)
