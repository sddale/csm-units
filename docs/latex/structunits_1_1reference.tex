The type for quantity references. 

This type is intended to be used in the quantity references definition\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{namespace\ }length\_references\ \{}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keyword}{inline}\ \textcolor{keyword}{constexpr}\ \textcolor{keyword}{auto}\ m\ =\ reference<dim\_length,\ metre>\{\};}
\DoxyCodeLine{\textcolor{keyword}{inline}\ \textcolor{keyword}{constexpr}\ \textcolor{keyword}{auto}\ km\ =\ reference<dim\_length,\ kilometre>\{\};}
\DoxyCodeLine{}
\DoxyCodeLine{\}}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keyword}{namespace\ }references\ \{}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keyword}{using\ namespace\ }length\_references;}
\DoxyCodeLine{}
\DoxyCodeLine{\}}

\end{DoxyCode}


Quantity references simplify quantity creation\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using\ namespace\ }units::isq::si::references;}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keyword}{auto}\ d\ =\ 123\ *\ m;}
\DoxyCodeLine{\textcolor{keyword}{auto}\ v\ =\ 70\ *\ (km\ /\ h);}

\end{DoxyCode}


Also, it is allowed to define custom quantity references from existing ones\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{constexpr}\ \textcolor{keyword}{auto}\ Nm\ =\ N\ *\ m;}
\DoxyCodeLine{\textcolor{keyword}{constexpr}\ \textcolor{keyword}{auto}\ mph\ =\ mi\ /\ h;}

\end{DoxyCode}


The following syntaxes are not allowed\+: {\ttfamily 2 / s}, {\ttfamily km $\ast$ 3}, {\ttfamily s / 4}, {\ttfamily 70 $\ast$ km / h}.
