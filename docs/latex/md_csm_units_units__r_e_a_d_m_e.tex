{\ttfamily mp-\/units} is a compile-\/time enabled Modern C++ library that provides compile-\/time dimensional analysis and unit/quantity manipulation. The basic idea and design heavily bases on {\ttfamily std\+::chrono\+::duration} and extends it to work properly with many dimensions.
