Detailed Description

template<std::intmax_t Num, std::intmax_t Den = 1>
struct units::ratio< Num, Den >

Provides compile-time rational arithmetic support.

This class is really similar to std::ratio. An important difference is the fact that the objects of that class are used as class NTTPs rather then a type template parameter kind.
