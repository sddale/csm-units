Detailed Description

template<Dimension D, UnitOf< D > U, Representation Rep = double>
class units::quantity< D, U, Rep >

A quantity.

Property of a phenomenon, body, or substance, where the property has a magnitude that can be expressed by means of a number and a measurement unit.

Template Parameters
Da dimension of the quantity (can be either a BaseDimension or a DerivedDimension)
Ua measurement unit of the quantity
Repa type to be used to represent values of a quantity
