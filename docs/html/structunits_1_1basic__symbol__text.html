Detailed Description

template<typename StandardCharT, std::size_t N, std::size_t M>
struct units::basic_symbol_text< StandardCharT, N, M >

A symbol text representation.

This class template is responsible for definition and handling of a symbol text representation. In the libary it is used to define symbols of units and prefixes. Each symbol can have two versions: Unicode and ASCI-only.

Template Parameters
StandardCharTCharacter type to be used for a Unicode representation
NThe size of a Unicode symbol
MThe size of the ASCII-only symbol
